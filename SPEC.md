# QClaw Agent Monitor - 智能体监控可视化界面

## 项目概述

为 QClaw/OpenClaw 打造的可视化监控 Dashboard，用于实时监控智能体及其子智能体的运行状态。

## 核心功能

### 1. 智能体树形展示
- 以树形结构展示主智能体及其所有子智能体
- 每个节点显示智能体名称、状态、运行时间
- 支持展开/折叠子节点

### 2. 运行状态监控
- **执行进度**: 当前步骤 / 总步骤
- **预计剩余时间**: 基于历史数据估算
- **资源使用情况**:
  - CPU 使用率
  - 内存占用
  - Token 消耗
  - API 调用次数

### 3. 卡住/锁死检测与报警
- 检测条件：
  - 无活动输出超过 5 分钟
  - 工具调用无响应
  - 状态长时间不变
- 触发方式：
  - 红灯闪烁
  - 声音提醒（可选）
  - 界面高亮标记

### 4. 恢复操作
- "继续任务"按钮：强制继续卡住的智能体
- "恢复"按钮：重启已失败的智能体
- 支持批量操作

## 技术栈

- **框架**: React + TypeScript
- **构建工具**: Vite
- **状态管理**: Zustand
- **HTTP 客户端**: 内置 fetch（调用 OpenClaw Gateway API）
- **样式**: CSS Modules

## API 接口

需要调用 OpenClaw Gateway 的 API：

| 接口 | 说明 |
|------|------|
| `GET /api/agents` | 获取所有智能体列表 |
| `GET /api/agents/:id` | 获取单个智能体详情 |
| `GET /api/agents/:id/subagents` | 获取子智能体列表 |
| `GET /api/sessions` | 获取会话列表 |
| `POST /api/agents/:id/resume` | 恢复智能体 |
| `POST /api/agents/:id/continue` | 继续任务 |

## 颜色方案

| 状态 | 颜色 | 说明 |
|------|------|------|
| 运行中 | `#22c55e` (绿色) | 正常执行 |
| 等待中 | `#f59e0b` (黄色) | 排队等待 |
| 卡住 | `#ef4444` (红色) | 需要干预 |
| 完成 | `#3b82f6` (蓝色) | 成功完成 |
| 未知 | `#6b7280` (灰色) | 初始/离线 |
