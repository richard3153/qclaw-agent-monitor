## 核心功能实现

1. **智能体树形展示** - 树形结构展示主智能体及子智能体
2. **状态监控** - 运行进度、预计剩余时间
3. **资源统计** - CPU、内存、Tokens、API 调用次数
4. **卡住检测** - 基于 5 分钟无活动自动检测
5. **报警机制** - 红灯闪烁 + 界面高亮
6. **恢复操作** - "继续任务"和"恢复"按钮

## 下一步

1. 在 Cursor 中打开项目后，安装依赖: `npm install`
2. 启动开发服务器: `npm run dev`
3. 访问 http://localhost:3000 查看

## 使用的技术栈

- React 18 + TypeScript
- Vite
- Zustand
- 内置 fetch 调用 OpenClaw Gateway API (http://localhost:16711)
